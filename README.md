# skygame_Log_monitoring_analysis
Sky游戏日志监听房主连接相关信息自动翻译为中文
从steam中打开游戏目录下找到“Sky.log”的文件，复制路径+文件名（例如H:\SteamLibrary\steamapps\common\Sky Children of the Light\Sky.log）即可进入监听状态
